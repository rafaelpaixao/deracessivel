Interface textual para DER (v3.0)

Protótipo v03.0


Utilize as teclas para navegar entre as opções. Use ENTER para confirmar uma opção, e ESC para voltar ao menu anterior. A qualquer momento utilize TAB para ir ao Menu Principal e CTRL para obter ajuda.
